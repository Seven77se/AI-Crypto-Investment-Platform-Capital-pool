## 📌 Project Introduction | 项目介绍

This project is a **blockchain-based investment platform** combined with AI data models and automated trading logic, designed for high-conversion traffic monetization.

该项目是一套完整的 **区块链智能投资系统**，结合 AI 数据模型与自动化收益逻辑，适用于海外流量变现与金融类平台搭建。

✔ 前后端分离架构
✔ 支持多语言（全球市场）
✔ 支持数字货币（USDT / USDC）
✔ 可快速部署上线运营

---

## 🛠 Tech Stack | 技术架构

* **Backend（后端）**: Java (Spring Boot)
* **Frontend Web（管理端）**: Vue2 + Element UI
* **Mobile App（用户端）**: UniApp（支持 H5 / App）
* **Database（数据库）**: MySQL
* **Architecture（架构）**: 前后端分离
